Storing objects (as in Javascript) in Scratch

object is a data structure which stores key value pairs. It can store references to other objects allowing for representation of trees.

Data

Each node of data requires these properties

key:    Key
type:   Data type of property, if is set to "Refrence" then value is a Reference to some object
value:  Value of property
next:   Reference to next property in object

Creating objects

Check if pool contains free node assign first property to free node set next to -1 return reference to first property

Accessing object properties

Jump to first property if given key matches property key -->return [type,value] else: -->jump to next return [None,None]

Deleting properties

set previous properties' next to -1 add this reference to pool

Deleting objects

use deleting properties to delete all properties
